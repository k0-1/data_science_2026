##  使いかた
1. **閲覧**: 上記の「資料リンク」をクリックしてMarkdownを読みます．
2. **実習**: 「Open in Colab」バッジをクリックすると，Google Colab上でコードを実行できます．

## 実習を始める前の注意
Colabで開いた直後の状態では**編集した内容が保存されません**.
自分の作業を保存するために，必ず以下の操作を行ってください．
1. 左上のメニューから[ファイル]を開く．
2. [ドライブにコピーを保存]をクリックする．
3. 新しく開いたタブ（名前に「～のコピー」とあるもの）で実習を進める．
